## Before You Begin

<div class="boiler-checklist">
  <span>Git installed locally</span>
  <span>A code editor ready to use</span>
  <span>Basic project requirements in mind</span>
  <span>A folder for your new project</span>
</div>

## Clone The Repository

```bash
git clone https://github.com/Nerdy-Technician/Boiler-Plates.git
cd Boiler-Plates
```

## Pick And Copy A Template

Browse the repository and choose the template closest to what you are building.
Then copy it into your project folder.

```bash
cp -r path/to/template /path/to/your/project
```

## Good Habits

<div class="boiler-card-grid">
  <article>
    <h3>Keep what you need</h3>
    <p>Remove unused files or dependencies so the copied template stays lean.</p>
  </article>
  <article>
    <h3>Commit the baseline</h3>
    <p>Make an initial commit after setup so your future changes are easy to review.</p>
  </article>
  <article>
    <h3>Contribute back</h3>
    <p>Open an issue or pull request when a template can be improved for everyone.</p>
  </article>
</div>
