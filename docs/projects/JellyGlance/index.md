## What It Does

<div class="store-steps">
  <article>
    <span>1</span>
    <h3>Observe</h3>
    <p>Track active Jellyfin streams, playback history, user activity, library health, and recently added media.</p>
  </article>
  <article>
    <span>2</span>
    <h3>Control</h3>
    <p>Manage JellyGlance users, Jellyfin Quick Connect roles, API keys, settings, backups, tasks, and webhooks.</p>
  </article>
  <article>
    <span>3</span>
    <h3>Integrate</h3>
    <p>Connect Arr apps and download clients for release calendars, queue monitoring, health checks, and event notifications.</p>
  </article>
</div>

## Highlights

<div class="jellyglance-feature-grid">
  <article>
    <strong>Jellyfin Native</strong>
    <span>Quick Connect login, synced users, libraries, artwork, activity, sessions, posters, and backdrops.</span>
  </article>
  <article>
    <strong>Useful Statistics</strong>
    <span>Top media, active users, watch-time trends, popular libraries, client usage, days, hours, and activity views.</span>
  </article>
  <article>
    <strong>Media Stack Aware</strong>
    <span>Sonarr, Radarr, Lidarr, Bazarr, qBittorrent, Transmission, Deluge, SABnzbd, and NZBGet integrations.</span>
  </article>
  <article>
    <strong>Operations Ready</strong>
    <span>Role permissions, API keys, scheduled tasks, logs, backups, download submissions, and webhook notifications.</span>
  </article>
